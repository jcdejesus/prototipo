Reto del Día: Calcular el Factorial

Dificultad: Fácil

Descripción del Problema:

Escribe una función que calcule el factorial de un número entero no negativo 'n'. El factorial de 'n' (representado como n!) es el producto de todos los enteros positivos menores o iguales que 'n'. Por ejemplo, 5! = 5 * 4 * 3 * 2 * 1 = 120.

Ejemplos:

factorial(0)  // 1
factorial(1)  // 1
factorial(5)  // 120
factorial(7)  // 5040

Consideraciones:

  • El número de entrada 'n' siempre será un entero no negativo.
  • Para 'n = 0', el factorial es 1 por definición.
