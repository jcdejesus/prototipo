Reto del Día: Minimum Window Substring

Dificultad: Medio

Descripción del Problema:

Dado un string s y un string objetivo t, encuentra el substring más corto en s que contenga todas las palabras (no caracteres) de t, en cualquier orden, al menos una vez cada una. Tanto s como t contienen palabras separadas por espacios. Debes devolver la subcadena original de s, no solo las palabras reordenadas. Si no existe tal substring, devuelve una cadena vacía.

Ejemplos:

Input:
                            s = "this is a test string that is just a test"
                            t = "test is a"
                            
                            Output:
                            "is a test"
                            
                            Input:
                            s = "practice makes perfect coding makes practice perfect"
                            t = "makes perfect practice"
                            
                            Output:
                            "practice makes perfect"

Consideraciones:

  • 1 <= len(s), len(t) <= 10^4 (¡sí, cadenas largas!)
  • Palabras pueden repetirse en t, y deben aparecer al menos ese número de veces en el substring.
  • La solución debe ser O(n) o lo más cercana posible
